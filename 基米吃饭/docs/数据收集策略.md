# 基米吃饭 - 数据收集策略

> **制定日期**：2026-06-13
> **参考对象**：Eat This Much
> **目标**：为 MVP 阶段收集高质量的食材营养数据

---

## 📊 数据源优先级

### 第一优先级（MVP 必须）

| 数据源 | 说明 | 数据量 | 格式 | 用途 |
|--------|------|--------|------|------|
| **USDA SR Legacy** | 标准参考数据 | ~8,000 种食物 | CSV | 核心营养数据 |
| **USDA Foundation Foods** | 基础食物数据 | ~2,000 种食物 | CSV | 补充营养数据 |

### 第二优先级（V1.0 推荐）

| 数据源 | 说明 | 数据量 | 格式 | 用途 |
|--------|------|--------|------|------|
| **中国食物成分表** | 中式食材数据 | ~2,000 种食物 | CSV/JSON | 中式食材补充 |
| **Open Food Facts** | 全球食品数据 | ~3,000,000 种 | CSV/JSON | 品牌食品补充 |

### 第三优先级（V1.5+ 可选）

| 数据源 | 说明 | 数据量 | 格式 | 用途 |
|--------|------|--------|------|------|
| **USDA Branded Foods** | 品牌食品数据 | ~300,000 种 | CSV | 品牌食品详细信息 |
| **餐厅数据** | 餐厅菜品数据 | 待定 | API/爬虫 | 外食选项 |

## 🎯 MVP 阶段数据收集计划

### 目标

- 收集 **500-1000 种** 常用食材的营养数据
- 数据格式统一，可直接用于模型训练
- 包含中英文食材名称

## 📥 数据收集步骤

### 步骤 1：下载 USDA SR Legacy 数据集

**下载链接**：
```
https://fdc-datasets/FoodData_Central_sr_legacy_food_csv_2018-04.zip
```

**文件大小**：约 30MB

**包含文件**：
- `food.csv` - 食物基本信息
- `nutrient.csv` - 营养素定义
- `food_nutrient.csv` - 食物营养数据
- `food_category.csv` - 食物分类

### 步骤 2：下载 USDA Foundation Foods 数据集

**下载链接**：
```
https://fdc-datasets/FoodData_Central_foundation_food_csv_2026-04-30.zip
```

**文件大小**：约 3.7MB

### 步骤 3：数据清洗和整合

**处理流程**：

```
原始 USDA 数据
    │
    ▼
┌─────────────────┐
│ 解析 CSV 文件   │  提取食物和营养数据
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 数据筛选        │  保留常用食材（去除冷门）
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 字段映射        │  映射到我们的数据格式
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 中文翻译        │  添加中文名称
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 数据验证        │  检查数据完整性和合理性
└─────────────────┘
    │
    ▼
┌─────────────────┐
│ 保存为 JSON     │  统一格式存储
└─────────────────┘
```

### 步骤 4：生成训练数据

**训练数据格式**：

```json
{
  "input": "蛋白质150g，脂肪60g，碳水200g",
  "output": "推荐食材：\n1. 鸡胸肉 200g（蛋白质40.0g，脂肪4.0g，碳水0.0g）\n2. 鸡蛋 3个（蛋白质18.0g，脂肪15.0g，碳水0.0g）\n3. 米饭 300g（蛋白质7.5g，脂肪2.4g，碳水72.0g）\n4. 西兰花 150g（蛋白质4.2g，脂肪0.6g，碳水5.1g）\n\n营养总计：蛋白质69.7g，脂肪22.0g，碳水77.1g\n总热量：805kcal"
}
```

**生成方法**：
1. 随机生成营养目标（蛋白质、脂肪、碳水）
2. 从食材库中选择合适的食材组合
3. 计算组合后的营养总量
4. 生成自然语言描述

## 📊 数据质量标准

### 必须满足

| 标准 | 说明 | 验证方法 |
|------|------|----------|
| **完整性** | 无空值 | 检查所有必填字段 |
| **准确性** | 营养数据合理 | 检查数值范围 |
| **一致性** | 格式统一 | 检查字段类型和单位 |
| **唯一性** | 无重复 | 检查 food_id 唯一 |

### 数值范围参考

| 营养素 | 单位 | 合理范围 |
|--------|------|----------|
| 蛋白质 | g/100g | 0-100 |
| 脂肪 | g/100g | 0-100 |
| 碳水 | g/100g | 0-100 |
| 热量 | kcal/100g | 0-900 |

## ⚠️ 注意事项

### 版权问题

- **USDA 数据**：公开免费使用，无版权限制
- **中国食物成分表**：版权属于中国 CDC，仅限个人学习使用
- **Open Food Facts**：Open Database License，可免费使用

### 数据更新

- USDA 数据每年更新一次
- 建议每季度同步一次最新数据
- 保留原始数据备份

### 翻译策略

- MVP 阶段：使用机器翻译 + 人工校对
- 后续阶段：建立食材名称对照表
- 优先翻译常用食材（Top 500）
