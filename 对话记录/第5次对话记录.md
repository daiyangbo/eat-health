#### 1. 更新 generate_train.py

修改 `format_output` 函数，添加总热量计算：

```python
def format_output(selected_foods):
    """
    格式化输出

    参数：
        selected_foods: 选择的食物列表

    返回：
        格式化的字符串
    """
    lines = ["推荐食材："]
    total_protein = 0
    total_fat = 0
    total_carbs = 0

    for i, item in enumerate(selected_foods, 1):
        food = item['food']
        serving = item['serving']
        protein = food['nutrition']['protein'] * serving / 100
        fat = food['nutrition']['fat'] * serving / 100
        carbs = food['nutrition']['carbs'] * serving / 100

        # 累加总量
        total_protein += protein
        total_fat += fat
        total_carbs += carbs

        lines.append(f"{i}. {food['name_en']} {serving}g（蛋白质{protein:.1f}g，脂肪{fat:.1f}g，碳水{carbs:.1f}g）")

    # 计算总热量（蛋白质4kcal/g，脂肪9kcal/g，碳水4kcal/g）
    total_calories = total_protein * 4 + total_fat * 9 + total_carbs * 4

    # 添加总计
    lines.append("")
    lines.append(f"营养总计：蛋白质{total_protein:.1f}g，脂肪{total_fat:.1f}g，碳水{total_carbs:.1f}g")
    lines.append(f"总热量：{total_calories:.0f}kcal")

    return "\n".join(lines)
```

**热量计算公式**：
- 蛋白质：4 kcal/g
- 脂肪：9 kcal/g
- 碳水化合物：4 kcal/g
- 总热量 = 蛋白质×4 + 脂肪×9 + 碳水×4
